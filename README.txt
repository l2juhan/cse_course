1. 라이브러리 설명
--------------------------------------------------------------------------------
컴퓨터공학과 교과목 정보를 조회할 수 있는 유틸리티 라이브러리입니다.
카테고리별(교양필수, 전공필수, 전공선택, 전공기초) 과목 조회 기능을 제공합니다.


2. 파일 구조
--------------------------------------------------------------------------------
cse_courses/
├── cse_courses.h      # 헤더 파일
├── cse_courses.c      # 라이브러리 소스 코드
├── test_static.c      # 정적 라이브러리 테스트 코드
├── test_dynamic.c     # 동적 라이브러리 런타임 링킹 테스트 코드 (dlopen)
├── libcse.a           # 정적 라이브러리
├── libcse.so          # 공유 라이브러리
├── Makefile           # 빌드 스크립트
└── README.txt         # 본 문서


3. 빌드 방법
--------------------------------------------------------------------------------

[방법 1: Makefile 사용]

    $ make all          # 전체 빌드 (정적/공유 라이브러리 + 테스트 프로그램)
    $ make static       # 정적 라이브러리만 빌드
    $ make shared       # 공유 라이브러리만 빌드
    $ make clean        # 빌드 파일 정리

[방법 2: 수동 빌드]

    # 정적 라이브러리 빌드
    $ gcc -c -Wall -fPIC cse_courses.c -o cse_courses.o
    $ ar rcs libcse.a cse_courses.o

    # 공유 라이브러리 빌드
    $ gcc -shared -fPIC -o libcse.so cse_courses.c

    # 정적 라이브러리 링킹 테스트 프로그램
    $ gcc -o test_static test_static.c -L. -lcse

    # 동적 라이브러리 런타임 링킹 테스트 프로그램
    $ gcc -o test_dynamic test_dynamic.c -ldl


4. 실행 방법
--------------------------------------------------------------------------------

[정적 라이브러리 테스트]
    $ ./test_static

[동적 라이브러리 테스트]
    $ ./test_dynamic
    (주의: libcse.so가 현재 디렉토리에 있어야 함)


5. 제공 함수
--------------------------------------------------------------------------------

void print_all_courses(void);
    - 설명: 전체 교과목 목록을 카테고리별로 출력
    - 매개변수: 없음
    - 반환값: 없음

void print_courses_by_category(const char* category);
    - 설명: 특정 카테고리의 과목만 출력
    - 매개변수: category - 카테고리명 ("교양필수", "전공필수", "전공선택", "전공기초")
    - 반환값: 없음

void print_categories(void);
    - 설명: 사용 가능한 카테고리 목록 출력
    - 매개변수: 없음
    - 반환값: 없음

const char* get_course_category(const char* course_name);
    - 설명: 특정 과목이 속한 카테고리 조회
    - 매개변수: course_name - 과목명
    - 반환값: 카테고리명 (찾지 못하면 NULL)

int get_course_count(const char* category);
    - 설명: 특정 카테고리의 과목 수 반환
    - 매개변수: category - 카테고리명
    - 반환값: 과목 수 (잘못된 카테고리면 -1)


6. 사용 예시
--------------------------------------------------------------------------------

#include <stdio.h>
#include "cse_courses.h"

int main(void) {
    // 전체 과목 출력
    print_all_courses();
    
    // 전공필수 과목만 출력
    print_courses_by_category("전공필수");
    
    // 특정 과목의 카테고리 조회
    const char* cat = get_course_category("시스템 프로그래밍");
    if (cat) {
        printf("시스템 프로그래밍은 %s 과목입니다.\n", cat);
    }
    
    // 전공선택 과목 수
    int count = get_course_count("전공선택");
    printf("전공선택 과목 수: %d\n", count);
    
    return 0;
}
